Программа, решающая задачу параллельного программирования "Производитель-потребитель".
Разрабатывалась в качестве курсового проекта для учёбы в 2018 году.

Система из двух параллельных процессов, включающая процесс-производитель и процесс-потребитель. Процессы взаимодействуют через некоторую обобщенную область памяти – буфер данных. В эту область процесс-производитель помещает очередные данные, а процесс-потребитель считывает очередные данные. Необходимо согласовать работу двух процессов при одностороннем обмене данными таким образом, чтобы удовлетворить следующим требованиям:
–	выполнять требования задачи взаимного исключения по отношению к критическому ресурсу – буферу данных;
–	учитывать состояние буфера сообщений, характеризующего возможность или невозможность посылки (принятия) очередного сообщения.
Процесс-производитель при попытке поместить очередное сообщение в полностью заполненный буфер должен быть блокирован. Попытка процесса-потребителя чтения из пустого буфера тоже должна быть блокирована.

Язык: Си.
Среда разработки: Visual Studio 2010.

Проект был сдан и завершен в январе 2018 года.
